🚀 PROMPT3D: Instant Neural Asset Studio
PROMPT3D aik advanced Generative AI platform hai jo natural language (text) ko high-fidelity 3D meshes mein convert karta hai. Ye project Large Language Models (LLMs) aur Neural Radiance Fields (NeRFs) ko mila kar user ko seconds mein 3D assets generate, inspect, aur export karne ki saholat deta hai.

✨ Key Features
Text-to-Mesh Synthesis: Text prompts se direct 3D geometry generate karna.

Real-time Viewport: Three.js aur WebGL use karte hue 3D model ko rotate aur inspect karna.

Multi-Format Export: Models ko .OBJ, .GLB, aur .STL formats mein download karna (Game engines aur 3D printing ke liye).

Cloud Storage: Firebase Cloud Firestore ke zariye users ke assets ko save rakhna.

Secure Authentication: Firebase Auth se login aur signup ka system.
